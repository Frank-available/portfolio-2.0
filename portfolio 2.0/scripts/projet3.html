Project 3: Designing a Customer Review System to Boost Sales

This project focuses on integrating and optimising a customer review system within an e-commerce website, aimed at increasing average basket value and building user trust during the purchasing process.

By highlighting positive feedback, user experiences, and displaying visual ratings (stars, scores), we successfully:

  • Encouraged faster purchasing decisions
  • Increased the conversion rate
  • Strengthened the site’s credibility
  • Motivated satisfied customers to recommend the site

System Illustrations

Technologies and Tools Used

  • HTML, CSS
  • JavaScript (for dynamic review display)
  • Tools: Trustpilot, Yotpo, Shopify App Reviews
